# README

Instructions:

* Ruby version - 3.3.0

* Rails version - 7.1.3.2

* Steps to run the app
  - clone repo
  - cd `ad_hbh_test`
  - run `gem install bundler`
  - run `bundle install`
  - setup `database.yml`
  - run `rails db:create db:migrate db:seed`
  - to run the app `rails s`
  - to run the specs(test cases) `rspec spec`
